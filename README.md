# dping-rs

dping-rs 是一个用 Rust 编写的简单高效的 ping 工具。

## 功能
- 支持 ICMP 协议的 ping
- 提供统计信息

## 使用方法
1. 克隆仓库：
   ```bash
   git clone https://github.com/chocolatedesue/dping-rs.git
   ```
2. 进入项目目录：
   ```bash
   cd dping-rs
   ```
3. 构建项目：
   ```bash
   cargo build --release
   ```
4. 运行程序：
   ```bash
   ./target/release/dping-rs
   ```
